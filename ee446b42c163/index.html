第一部分：积累运用(72分)


一、我会看拼音写词语。(10分)



二、我会选择恰当的字填写在括号里。(8分) 



三、一字组多词。(8分)

劳(　　　　　　)(　　　　　　)　　　　能(　　　　　　)(　　　　　　)

景(　　　　　　)(　　　　　　)　　　　留(　　　　　　)(　　　　　　)

四、我会连一连。(8分)



五、我会选择正确的答案，把字母填写在括号里。(18分)

1．下列加点字的读音有误的一项是(　　　　)

A．马匹(pǐ) 　　一株(zhū)

B．工具(jù)　　荆棘(jīng)

C．曾经(céng) 　　的确(de)

D．建筑(zhù)　　顺序(shùn)

2．下面的字与“狞”字读音相近的一项是(　　　　)

A．狂 　　　　 B．泞 　　　　 C．叮  　　　D．狩

3．“年迈的大娘”中的“迈”是什么意思？(　　　　)

A．抬起腿来跨步  　　B．老  　　C．远行  　　D．豪放

4．下列词语中意思相反的一项是(　　　　)

A．寻找—寻觅　 　　　　　B．仔细—认真　 

C．特别—特殊　 　　　　　D．难受—舒服

5．给加点词选择正确的解释。

【经过】A.通过(处所、时间、动作等)；B.经历。

(1)请说说你探险的经过。(　　　　)

(2)经过操场时，我看见李明正在打球。(　　　　)

【自然】A.自然界；B.表示理所当然。

(3)动物和植物都是大自然的成员。(　　　　)

(4)她平时上课认真听讲，回家及时复习，自然能考出好成绩。(　　　　)

6．(　　　　)被人们誉为“人类灵魂的工程师”。

A．建筑师  　　B．理发师　　　　C．工程师 　　 D．教师

六、我会照样子完成句子练习。(6分)

1．例：他拾起一根枝条。　他拾起一根又长又细的枝条。

树上挂满了柿子。　______________________________________

2．例：田野，葱葱绿绿的，像一片柔软的绿毯。

(1)小河，____________________，_________________________。

(2)________，________________，_________________________。

七、我会根据课本内容完成练习。(14分)

1．生活中，那些本来就很困难的人更需要我们的帮助，正所谓“与其________________，不如________________”。

2．一向热心帮助别人的小明遇到了困难，大家都乐意帮助他。这真是“平时________________，急时________________”。

3．在无私奉献的同时，我们自己也获得了快乐，因为“予人玫瑰，________________________”。

4．雷锋叔叔帮助了迷路的孩子和____________________________，他是一个________________________________的人。


第二部分：阅读欣赏(13分)


八、我会阅读短文，并完成练习。(2＋2＋2＋4＋3＝13分)

小黑熊摔(shuāi)伤了腿，一个人，好孤单。

天蓝蓝的，阳光金灿灿的，空气多么好！小黑熊说：“小鸟小鸟，你能帮我瞧瞧我的好朋友都在做什么吗？”

小鸟回来了，告诉小黑熊：“小棕(zōnɡ)熊在摘苹果，小浣(huàn)熊在烤葡萄干面包，小白熊在锯木头，小树袋熊在采花。”“朋友们都在做自己的事，可是我……什么也没法做。”小黑熊很难过。

“叮咚——叮咚——”谁来了？呀，小棕熊、小浣熊、小白熊……朋友们都来了！

小棕熊带来了一篮苹果，小浣熊拿出香喷喷的面包，小白熊带来了一把漂亮的小摇椅，小树袋熊送给小黑熊一个美丽的花环。大家齐声说：“小黑熊，祝你快点好起来啊！”

1．这篇短文一共有________个自然段。

2．(　　　　)摔伤了腿，一个人，好孤单。

A．小黑熊　　 B．小棕熊　　　　C．小浣熊　　D．小白熊

3．小鸟告诉小黑熊，小树袋熊在做什么？(　　　　)

A．摘苹果  　　B．烤葡萄干面包   C．锯木头   D．采花

4．小黑熊的朋友们给他送来了什么礼物？

小棕熊送的是(　　　　)，小浣熊送的是(　　　　)，小白熊送的是(　　　　)，小树袋熊送的是(　　　　)。

A．香喷喷的面包  　　　　　　 B．一篮苹果

C．一个美丽的花环 　　　　　 D．一把漂亮的小摇椅

5．小黑熊收到朋友们的礼物后会说些什么？想一想，写一写。
